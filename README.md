# VueModel

## 介绍
vue前端项目模版（自用）

## 软件架构
- 前端Vue项目模版，
- 使用Vite构建项目
- 使用Pinia进行全局用户登录态管理
- 使用axios进行全局请求调用
- 使用openapi根据文档快速生成请求函数
- 使用Vue Router进行全局路由管理
- 使用arco design制作基础布局页面和登录注册页面


## 安装教程

1.克隆或下载本仓库

2.使用
>pnpm install

或其他node包管理器安装项目依赖
